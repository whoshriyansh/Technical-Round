Create a minimal AI chat interface that lets the user type a prompt, send it to the OpenAI API, and display the model’s response.

Requirements

1. Basic Chat UI
   A. text input field to enter messages
   B. “Send” button
   C. visible chat history area showing both user and AI messages

2. API Integration

3. Functional Requirements
   When the user sends a message:
   Add the user message to chat history
   Disable the send button (loading state)
   Fetch the AI response from API (or mock)
   Display AI response below the user message
   Clear input after sending
